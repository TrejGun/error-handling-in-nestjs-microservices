# Docker

### Build and Start all in Docker
```sh
docker-compose -f docker-compose.yml up --build
```
### Stop all in Docker
```sh
docker-compose -f docker-compose.yml down -v
```
### Restart all in Docker
```sh
docker-compose -f docker-compose.yml up
```
